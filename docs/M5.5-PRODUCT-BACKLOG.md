# Vela M5.5 产品补全计划

## 阶段定义
M5 已收口（2026-03-22）。M6（Electron Builder 打包）延后。
在 M5 和 M6 之间插入 **M5.5 产品补全**，专门清理产品层缺口 + 提升首次体验质感。

使用 **Gemini 3.1 Pro Preview**（lemonapi 代理）生图：
- K Studio logo 高清版（替换豆包水印版）

## 施工顺序（已锁定）

### Phase 1：核心能力补全

#### M5.5-T1 ASR 语音识别接入
- 优先级：P0
- 状态：TODO
- 说明：当前 `asr.enabled: false, provider: "placeholder"`。语音按钮按下无法语音转文字。

### Phase 3：视觉氛围

#### M5.5-T11 Avatar 场景背景
- 优先级：P1
- 状态：TODO
- 说明：当前 VRM 模型背景空白单薄，需要场景插画
- 方案：**静态插画（A）+ 轻粒子点缀（B）**
  - 主体：二次元风格房间/窗台插画，铺在 VRM 模型后层
  - 日/夜两套：根据 time-provider 自动切换（白天窗台明亮 / 夜晚台灯暖光）
  - 轻粒子：极少量浮动光斑（Three.js / CSS），不抢主角
- 资产：Gemini 3.1 Pro 生成，提供人物参考图 + 项目背景描述
- 验收：avatar 有场景感，日/夜自动切换，不影响 VRM 模型渲染性能

#### M5.5-T12 环境 BGM
- 优先级：P1
- 状态：TODO
- 说明：伴侣产品氛围音乐
- 设计：
  - 风格：轻氛围（lo-fi / soft piano / ambient），不抢 TTS 语音
  - 日/夜两套 BGM，配合时间感知切换
  - **必须有开关和音量滑块**（防公共场合社死）
  - TTS 播放时 BGM 自动降到 20% 音量（ducking）
  - Electron 环境可绕过浏览器 autoplay 限制
  - 偏好持久化到 local-store
- 曲源：免费可商用 BGM 库（Pixabay Audio / freepd.com / 用户自备）
- 验收：启动有 BGM → TTS 时自动降音 → 音量可调 → 可关闭 → 重启记忆偏好

### Phase 4：安全网

#### M5.5-T3 错误处理 UI
- 优先级：P0
- 状态：TODO
