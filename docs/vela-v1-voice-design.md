# Vela_v1 Voice Design — 甜妹线（第一版）

## 目标
先做一条**年轻、甜、但不夹**的中文女性陪伴声线，作为 `Vela_v1` 的前期默认声音。

当前 M2 施工上下文里，系统默认 `voiceId` 先锁定为 `Chinese (Mandarin)_Sweet_Lady`；
这份文档保留的是后续自定义 `Vela_v1` 声线设计目标，不代表 M2 默认声线已经切换。

这条线的目标不是夸张二次元配音，也不是播音腔、客服腔，而是：
- 听起来是年轻女性
- 有亲近感和陪伴感
- 甜，但不过度发嗲
- 清晰自然，不含糊
- 温柔、有耐心、愿意接住人
- 不廉价，不浮夸，不机械

## 设计约束
- 不要过亮、过尖、过嗲
- 不要新闻播报感
- 不要成熟御姐感
- 不要明显娃娃音
- 不要强表演腔
- 不要故意做成“动漫夸张角色配音”

## 推荐 prompt（V1）
年轻女性的中文声音，二十岁出头，声线甜润但克制，温柔亲近，带一点轻柔的少女感和陪伴感。说话自然、清晰、放松，不夹、不做作，不像播音员，也不像客服。整体气质是温暖、安静、愿意认真听人说话，能给人被接住的感觉。语速中等偏慢一点，咬字清楚，尾音自然，情绪表达细腻，但不过度夸张。像一个会陪你深夜聊天、温柔回应你、让人放松下来的年轻女生。

## preview_text（V1）
晚上好，我在。你可以慢慢说，我会接着你。先听听这条声音，看看她像不像你想要的 Vela。

## 备用收紧词
如果 V1 太平、太路人，可以增强：
- 更轻柔一点
- 更有少女感一点
- 更有陪伴感一点

如果 V1 太夹、太甜腻，可以收紧：
- 更自然一点
- 更克制一点
- 更像真实聊天，不要表演感
